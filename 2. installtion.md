#### How to install in Linux OS?

```
yum install git -y
git --version  
```

#### install on Debian:

```
sudo apt-get update
sudo apt-get install git

```

#### config steps:

```
git config --global user.name "your user name"
git config --global user.email "your email"

# to check the config list
git config --list

# to see the config file
ls -a

# or
~/.gitconfig

```
